# ubunopy

(Ubu)ntu based Docker image with (No)de.js and (Py)thon inside.

## Info

* Based on [bitbar/ubuno](https://github.com/bitbar/ubuno) which has:

  * Ubuntu LTS (22.04)
  * Node.js (14)
  * [learn more](https://github.com/bitbar/ubuno)

* Python 3 + pip
* Python 2.7 + pip
