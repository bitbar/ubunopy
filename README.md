# ubunopy

(Ub)untu based Docker image with (No)de.js and (Py)thon inside.

## Info

* Based on Ubuntu LTS (18.04)
* Node.js LTS (10)
* Python 3 + pip (Python 2.7 installed by Node.js installer)
* other tools:

  * curl
  * wget
  * git
  * gcc (helpful for some node packages - eg. node-gyp)
  * tree
